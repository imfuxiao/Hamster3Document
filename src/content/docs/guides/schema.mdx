## 输入方案目录结构

<FileTree>
- RimeSharedSupport/
  - opencc/
  - default.yaml
  - ...
- RimeUserData/
  - rime-ice/
  - rime-wanxiang/
</FileTree>

- `RimeSharedSupport`: 对应 rime 引擎的 `shared_data_dir` 目录。
    > 注意：元书 RimeSharedSupport 目录不包含 PC 上内置的输入方案，仅包含一些 default.yaml，opencc 等基础配置文件。
    > 如果你的方案依赖一些 PC 上内置的输入方案，请一起打包上传至元书
- `RimeUserData`: 存放用户输入方案的根目录，你可以在此目录下为不同的方案创建各自的子目录。
    如示例中的雾凇对应`rime-ice`目录，万象对应`rime-wanxiang`目录。
    
    元书支持将`RimeUserData`下任意子目录作为 rime 引擎的 `user_data_dir` 目录。
    
    你可以通过「输入方案」右上角菜单 `...` 下的「方案目录切换」功能，切换当前输入方案目录。

## 如何导入 RIME 的输入方案

元书支持多种方式导入输入方案，您可以根据实际情况选择最方便的方法。

### 方案一：通过 WiFi 传输

这种方式适合在电脑和手机处于同一局域网时使用，无需数据线即可传输。

1. 在元书中打开「WiFi 文件传输」功能，会显示一个网址
2. 在电脑浏览器中打开这个网址
3. 双击「RimeUserData」文件夹进入
4. 将电脑上的方案文件直接拖拽到浏览器窗口中，文件会自动上传
5. 上传完成后，打开元书的「输入方案」页面，点击右上角菜单选择「方案目录切换」，选则刚才上传的文件夹

<Aside type="tip">
**重要提示：**
- 建议在「RimeUserData」目录下创建子文件夹来存放方案。例如导入「雾凇拼音」方案时，可以创建「rime-ice」文件夹，然后将方案文件放入其中
- 使用此功能请保持手机长亮，且不要将应用切换到后台，否则传输会中断
</Aside>

### 方案二：通过压缩包导入

这种方式适合需要分享方案文件的场景。

1. 将方案文件压缩为 zip 格式（请确保 zip 文件使用 UTF-8 编码）
2. 通过任意方式（微信、QQ、AirDrop 等）将文件传到手机
3. 使用元书打开该文件，元书会自动导入方案
    
    - 也可以在元书的「输入方案」页面中，点击右上角...菜单，选择「导入方案」，在目录里找刚才保存到手机上的 zip 文件
    
4. 导入完成后，打开「输入方案」页面，点击右上角菜单选择「方案目录切换」，选则刚导入的文件夹

### 方案三：从网络下载

这种方式适合直接从 GitHub 等网站下载方案。

1. 打开元书的「输入方案」页面，点击右上角菜单，选择「下载方案」
2. 点击右上角的 ➕ 按钮，在弹出窗口中填写下载地址和文件名，点击「保存」
3. 在列表中找到刚才添加的项目，点击右侧的绿色下载按钮
5. 等待下载完成后，进入「输入方案」页面，点击右上角菜单选择「方案目录切换」，选择刚下载的文件夹
