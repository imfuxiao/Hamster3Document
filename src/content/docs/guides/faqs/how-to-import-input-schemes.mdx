---
title: 如何导入输入方案？
---

import { Aside } from '@astrojs/starlight/components';

元书支持多种方式导入输入方案，您可以根据实际情况选择最方便的方法。

## 方案一：通过 WiFi 传输

这种方式适合在电脑和手机处于同一局域网时使用，无需数据线即可传输。

1. 在元书中打开「WiFi 文件传输」功能，会显示一个网址
2. 在电脑浏览器中打开这个网址
3. 双击「RimeUserData」文件夹进入
4. 将电脑上的方案文件直接拖拽到浏览器窗口中，文件会自动上传

<Aside type="tip">
**重要提示：**
- 建议在「RimeUserData」目录下创建子文件夹来存放方案。例如导入「雾凇拼音」方案时，可以创建「rime-ice」文件夹，然后将方案文件放入其中
- 上传完成后，打开元书的「输入方案」页面，点击右上角菜单选择「方案目录切换」，选中刚才上传的文件夹即可
- 使用此功能请保持手机长亮，且不要将应用切换到后台，否则传输会中断
</Aside>

## 方案二：通过压缩包导入

这种方式适合需要分享方案文件的场景。

1. 将方案文件压缩为 zip 格式（请确保使用 UTF-8 编码）
2. 通过任意方式（微信、QQ、AirDrop 等）将文件传到手机
3. 使用元书打开该文件，会自动解压并导入
4. 导入完成后，打开「输入方案」页面，点击右上角菜单选择「方案目录切换」，选中导入的文件夹

## 方案三：从网络下载

这种方式适合直接从 GitHub 等网站下载方案。

1. 打开元书的「输入方案」页面，点击右上角菜单，选择「下载方案」
2. 点击右上角的 ➕ 按钮
3. 在弹出窗口中输入方案的下载地址和文件名，点击「保存」
4. 在列表中找到刚才添加的项目，点击右侧的绿色下载按钮
5. 等待下载完成后，进入「输入方案」页面，点击右上角菜单选择「方案目录切换」，选中下载的文件夹
