---
title: 如何快速将仓迁移到元书中使用？
---

import { Aside } from '@astrojs/starlight/components';

元书支持导入仓中的备份文件。

您可以通过以下步骤将仓中的备份文件导入到元书中使用：


1. 在仓的备份功能中新建一个备份包。
2. 打开元书，进入“备份与回复”页面，选择右上角的“导入仓输入法备份”。
3. 在打开页面中选择刚才创建的备份包。（注意入口在：我的手机-Hamster-backups下）。
4. 如果一切正常，那么仓中的方案文件，剪贴板，常用语，脚本等数据会导入到元书中。
5. 导入完成后，需要进入「输入方案」页面，点击右上角菜单中的「方案目录切换」，选择刚才导入的「Hamster-backups」文件夹。


<Aside title="注意" type="danger">
    元书中的皮肤文件与仓中的皮肤文件不兼容，导入时不会迁移皮肤。
</Aside>
